# pi-file-todos

A pi package that provides the `file-todos` skill for managing a file-based todo system in `todos/`.

## Included skill

- `file-todos` — todo naming conventions, lifecycle workflow, dependency tracking, triage, and work-log standards

## Installation

### From local path

```bash
pi install /path/to/playbook/packages/pi-file-todos
```

### Project-local

```bash
pi install -l /path/to/playbook/packages/pi-file-todos
```

## Package structure

```text
pi-file-todos/
├── package.json
├── README.md
└── skills/
    └── file-todos/
        ├── SKILL.md
        └── assets/
            └── todo-template.md
```
