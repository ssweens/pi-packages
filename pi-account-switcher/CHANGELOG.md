# Changelog

All notable changes to this project will be documented in this file.

## [0.2.4] - 2026-05-30
### Changed
- Renamed the npm package from `@hieplp/pi-account-switcher` to `@ssweens/pi-account-switcher` so future publishes go to the `@ssweens` scope.
- Updated package metadata to point at the `ssweens/pi-packages` monorepo (`repository`, `homepage`, `bugs`).
- Updated install docs to recommend `pi install npm:@ssweens/pi-account-switcher`.

## [0.2.3] - 2026-05-28
### Added
- Added `Ctrl+Shift+C` keyboard shortcut to open the account selector and activate the chosen account (equivalent to `/accounts:list`).

### Updated
- Documented the shortcut in `README.md` and `USAGE.md`.
