- `recorded.<bridge-resources>` -- the entries to record in state.json (Phase 5 reads these)

- Foreign file with both gates passing → if owned by a different `(marketplace, plugin)` per the agents-index, throws `AgentOwnershipConflictError` (AG-9). If not in the index at all, treated as untracked legitimate-looking content; bridge MUST refuse to touch and surface via `failed[]`.
