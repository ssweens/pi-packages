```yaml
name: CI

# D-01: single-Node-version matrix. Node 24 only -- the latest stable LTS line
# at the time of Phase 1 planning. Justification (per CONTEXT.md):
#   - Node 24 satisfies write-file-atomic@^8's `^22.22.2 || ^24.15.0 || >=26.0.0`
#     engine floor.
#   - Node 24 supports native TS strip (no `tsx`, no `--experimental-strip-types`).
#   - Multi-version matrix maintenance overhead is not justified for V1.
#
# Note: ROADMAP.md Phase 1 success criterion 5 originally specified Node
# 22/24/26. D-01 supersedes that with single-version Node 24. If a future
# audit reopens this, add `node-version: [22, 24, 26]` and a `strategy.matrix`
# block; the rest of the workflow is matrix-ready.

jobs:
  check:
    name: npm run check (Node 24)
    runs-on: ubuntu-latest
    timeout-minutes: 15
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Setup Node 24
        uses: actions/setup-node@v4
        with:
          node-version: "24"
          cache: "npm"

      - name: Install dependencies
        run: npm ci

      - name: Run full check pipeline
        # NFR-6: typecheck + ESLint + Prettier + node:test
        run: npm run check
```

Notes:
- `actions/checkout@v4` and `actions/setup-node@v4` are the current LTS versions of the canonical actions (verified at the time of Phase 1 planning -- 2026-05-09).
- `cache: "npm"` enables npm cache restore across runs without configuring it manually.
- `npm ci` (not `npm install`) is the right call for CI: it installs from `package-lock.json` exactly, fails on drift, and skips the lockfile rewrite step.
- The `timeout-minutes: 15` is generous; Phase 1's surface is small (`npm run check` runs in well under a minute on a clean Node 24 node) but `npm ci` cold-cache pulls can hit network latency.
- Trigger pattern (`on.push.branches.features/**`) matches the current branch convention so PRs from feature branches run CI.

Do NOT add cron schedules, deploy steps, release workflows, or coverage uploads. Phase 1's CI scope is "npm run check on Node 24 -- nothing more, nothing less."
