    Create `extensions/pi-claude-marketplace/edge/handlers/plugin/reinstall.ts`.

    Concrete file contents and behavior:
    - Import `reinstallPlugins` from `../../../orchestrators/plugin/reinstall.ts`.
    - Import `errorMessage` from `../../../shared/errors.ts`.
    - Import `notifyError` from `../../../shared/notify.ts`.
    - Import `parseArgs` from `../../args.ts`.
    - Import `splitPluginMarketplaceRef` from `./shared.ts`.
    - Define the usage constant exactly:
      ```typescript
      const USAGE =
        "Usage: /claude:plugin reinstall [<plugin>@<marketplace> | @<marketplace>] [--scope user|project] [--force]";
      ```
    - Export `makeReinstallHandler(pi: ExtensionAPI): (args: string, ctx: ExtensionCommandContext) => Promise<void>`.
    - Inside the handler, parse with `parseArgs(args)` inside `try/catch`; on parse error call `notifyError(ctx, `${errorMessage(err)}\n${USAGE}`)` and return.
    - Walk `parsed.positional` and extract a boolean `force` plus non-flag positionals:
      - If token is exactly `"--force"`, set `force = true` and do not include it as a ref.
      - If token starts with `"--"` and is not `"--force"`, call `notifyError(ctx, `Unknown option: "${token}".\n${USAGE}`)` and return.
      - Otherwise push token into `refs`.
    - If `refs.length > 1`, call `notifyError(ctx, `Too many arguments.\n${USAGE}`)` and return.
    - If `refs.length === 0`, set `target = { kind: "all" }`.
    - If `refs[0]` starts with `"@"` and `refs[0].length > 1`, set `target = { kind: "marketplace", marketplace: refs[0].slice(1) }`.
    - Otherwise call `splitPluginMarketplaceRef(refs[0])`; if it returns undefined call `notifyError(ctx, USAGE)` and return; otherwise set `target = { kind: "plugin", plugin: ref.plugin, marketplace: ref.marketplace }`.
    - Call:
      ```typescript
      await reinstallPlugins({
        ctx,
        pi,
        cwd: ctx.cwd,
        target,
        ...(parsed.scope !== undefined && { scope: parsed.scope }),
        ...(force && { force: true }),
      });
      ```
    - Do not accept `--force=true`, `-f`, or arbitrary `--*` options.

    Create `tests/edge/handlers/plugin/reinstall.test.ts` with hermetic HOME/cwd setup like `tests/edge/handlers/plugin/update.test.ts`.
    Required test names and assertions:
    - `shim :: bare reinstall with no positional calls reinstallPlugins target all` asserts notification contains `No plugins installed.` on a fresh state.
    - `shim :: @marketplace form calls reinstallPlugins marketplace target` asserts a missing marketplace produces one error notification containing `mymkt` and `not found`.
    - `shim :: plugin@marketplace form calls reinstallPlugins plugin target` asserts a missing marketplace produces one error notification containing `mymkt` and `not found`.
    - `shim :: --scope works before and after reinstall ref` invokes `--scope project` and `myplug@mymkt --scope project` without usage errors; the second assertion should contain `Marketplace "mymkt" not found in project scope.`.
    - `shim :: --force works before and after reinstall ref` creates an installed plugin with foreign previous agent content and proves `reinstall ... --force` reaches a successful or save-failure rollback-protected force path instead of the default `foreign previous content` error.
    - `shim :: invalid ref unknown flag and extra positionals emit reinstall usage` asserts `no-at-sign`, `--bogus`, `a@mp b@mp`, and `--force=true` each produce a notification matching `Usage: /claude:plugin reinstall` and do not perform a successful reinstall.
