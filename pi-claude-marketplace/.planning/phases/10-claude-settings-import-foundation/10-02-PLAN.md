## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-04 | Tampering | enabled plugin values | mitigate | Accept only exact boolean `true`; reject strings/numbers/objects with diagnostics. |
| T-10-05 | Spoofing | plugin ref keys | mitigate | Require exactly one `@` and non-empty plugin/marketplace halves before producing refs. |
| T-10-06 | Denial of Service | malformed entries | mitigate | Non-throwing parser keeps valid entries processing after bad entries. |
</threat_model>
